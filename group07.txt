Next steps
==========

 * find a typo in this file
 * fix it
 * save file
 * tell git about the change: git add FILENAME
 * commit the change: git commit -m "fixed typo in FILENAME"
 * push the commit to github: git push
 * go to your fork https://github.com/YOURUSERNAME/pull-request-please
 * hit 'new pull request', review your changes and
   create a pull request against andrenarchy/pull-request-please
 * wait until pull request is reviewed and merged
 * now you know how git works
